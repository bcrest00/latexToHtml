7.3   Funciones polinómicas

7.3.1   Modelo de función polinómica

Analiza y representa la función y = 2x2 − x4 4

  • 1. Tipo de función: polinómica.
  • 2. Dominio: por ser una función polinómica, es toda la recta real ℝ
    Dom(f) = ℝ = (−∞, +∞).
  • 3. Continuidad: por ser polinómica, es toda la recta real ℝ.
  • 4. Periodicidad: no es periódica porque las funciones polinómicas nunca lo son.
  • 5. Simetrías: f(−x) = 2(−x)2 − (−x)4 4 = 2x2 − x4 4
    Se observa que f(−x) = f(x) → función par → simetría respecto del eje Y.
  • 6. Asíntotas: las funciones polinómicas no tienen asíntotas.
  • 7. Corte con los ejes.
    • Eje X: 2x2 − x4 4 = 0 → x = 0, raíz doble; x1 = 22, x2 = −22, raíces simples.
      Se obtienen los puntos O(0, 0), A(−22, 0), B(22, 0)
    • Eje Y: es el punto O(0, 0)
    • Signo: Si x = 1 → f(1) = 2 − 1 4 = 7 4 > 0 (+)
  • 8. Máximos y mínimos:
    f′(x) = 4x − x3 → 4x − x3 = 0 → x 1 = 0,x2 = −2,x3 = 2, raíces simples.
    f(x) = 2x2 − x4 4
    • f(−2) = 4 → C(−2, 4)
    • f(0) = 0 → O(0, 0)
    • f(2) = 4 → D(2, 4)

    f″ ⁡(x) = 4 − 3x2

    • f″ ⁡(−2) = −8 < 0 → C(−2, 4) máximo relativo.
    • f″ ⁡(0) = 4 > 0 → O(0, 0) mínimo relativo.
    • f″ ⁡(2) = −8 < 0 → D(2, 4) máximo relativo.

    Monotonía:
    f′(x) = 4x − x3 → Si x = 1 → f′(x) = 4 − 1 = 3 > 0 (+)

  • 9. Puntos de inflexión:
    f″ ⁡(x) = 4 − 3x2 → 4 − 3x2 = 0 → x 1 = −23 3 , x2 = 23 3 raíces simples.
    f(x) = 2x2 − x4 4
    • f(−23 3 ) = 20 9 → E(−23 3 , 20 9 )
    • f(23 3 ) = 20 9 → F(23 3 , 20 9 )

    f‴ ⁡(x) = −6x

    • f‴ ⁡(−23 3 ) = 43≠0 → E(−23 3 , 20 9 ), punto de inflexión.
    • f‴ ⁡(23 3 ) = −43≠0 → F(23 3 , 20 9 ), punto de inflexión.
  • 10. Curvatura:
    f″ ⁡(x) = 4 − 3x2 → Si x = 0 → f″ ⁡(0) = 4 > 0 (+)

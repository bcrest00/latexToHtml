3.8   Funciones trigonométricas

3.8.1   Modelo de función trigonométrica

Analiza y representa la función y = 3 sin ⁡ 2x

  • 1. Tipo de función: trigonométrica.
  • 2. Dominio: las funciones seno y coseno están definidas en todos los números reales ℝ.
    Dom(f) = ℝ = (−∞, +∞)
  • 3. Continuidad: el seno y coseno son continuas en toda la recta real ℝ.
  • 4. Periodicidad: es periódica en, de período 2π 2 = π
    Solo la estudiaremos en el primer período positivo [0,π)
  • 5. Simetrías: f(−x) = 3 sin ⁡ (−2x) = −3 sin ⁡ (2x)
    Se observa que f(−x) = −f(x) → función impar → simétrica respecto al origen de coordenadas O(0, 0).
  • 6. Asíntotas:
    • Verticales: no tiene.
    • Horizontales: no tiene.
    • Oblicuas: no tiene.
  • 7. Corte con los ejes:
    • Eje X: 3 sin ⁡ (2x) = 0 → sin ⁡ (2x) = 0 → x1 = 0, x2 = π 2, raíces simples; Se obtienen los puntos O(0, 0); A(π 2, 0).
    • Eje Y: es el punto O(0, 0)
    • Signo: Si x = π 4 → f(π 4) = 3 ⋅ sin ⁡ (2π 4 ) = 3 sin ⁡ (π 2) > 0 (+)
  • 8. Máximos y mínimos:
    f(x) = 3 sin ⁡ (2x)
    • f(π 4) = 3 sin ⁡ (π 2) = 3 → B(π 4, 3)
    • f(3π 4 ) = 3 sin ⁡ (3π 2 ) = −3 → C(3π 4 ,−3)

    f″ ⁡(x) = −12 sin ⁡ (2x)

    • f″ ⁡(π 4) = −12 < 0 → B(π 4, 3), máximo relativo.
    • f″ ⁡(3π 4 ) = 12 > 0 → C(3π 4 ,−3), mínimo relativo.

    Monotonía:
    f′(x) = 6 cos ⁡ (2x) → Si x = π 2 → f′(π 2) = 6 cos ⁡ (π) = −6 < 0 (-)

  • 9. Puntos de inflexión:
    f″ ⁡(x) = −12 sin ⁡ (2x) → sin ⁡ (2x) = 0
    • 2x = 0 → x = 0, raíz simple.
    • 2x = π → x = π 2, raíz simple.

    f(x) = 3 sin ⁡ (2x)

    • f(0) = 3 sin ⁡ (0) = 0 → O(0, 0)
    • f(π 2) = 3 sin ⁡ (π) = 0 → A(π 2, 0)

    f‴ ⁡(x) = −24 cos ⁡ (2x)

    • f‴ ⁡(0) = −24 cos ⁡ (0) = −24≠0 → O(0, 0), punto de inflexión.
    • f‴ ⁡(π 2) = −24 cos ⁡ (π) = 24≠0 → A(π 2, 0), punto de inflexión.

    Curvatura:
    f″ ⁡(x) = −12 sin ⁡ (2x) → Si x = π 4 → f″ ⁡(π 4) = −12 sin ⁡ (π 2) = −12 < 0 (-)

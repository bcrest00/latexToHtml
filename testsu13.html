3.3   Funciones racionales

3.3.1   Modelo de función racional

Analiza y representa la función y = x3 x2 − 1

  • 1. Tipo de función: racional.
  • 2. Dominio: por ser una función racional, hay que excluir las raíces del denominador: x2 − 1 = 0 → x2 = 1 → x 1 = −1, x2 = 1.
    Dom(f) = ℝ −{−1, 1} = (−∞,−1) ⋃ ⁡ (−1, 1) ⋃ ⁡ (1, +∞)
  • 3. Periodicidad: no es periódica porque las funciones racionales nunca lo son.
  • 4. Simetrías: f(−x) = (−x)3 (−x)2 − 1 = −x3 x2 − 1 = − x3 x2 − 1
    Se observa que f(−x) = −f(x) → función impar → simetría respecto al origen O(0, 0)
  • 5. Asíntotas:
    • Verticales: son las raíces del denominador, x = −1, x = 1
      Posición de la curva respecto a las asíntotas verticales:
      lim ⁡ x→−1−( x3 x2 − 1) = (−1−)3 (−1−)2 − 1 = −1 0+ = −∞

      lim ⁡ x→−1+( x3 x2 − 1) = (−1+)3 (−1+)2 − 1 = −1 0− = +∞

      lim ⁡ x→1−( x3 x2 − 1) = (1−)3 (1−)2 − 1 = 1 0− = −∞

      lim ⁡ x→1+( x3 x2 − 1) = (1+)3 (1+)2 − 1 = 1 0+ = +∞

    • Horizontales: no tiene.
    • Oblicuas: y = x
      Posición de la curva respecto de la asíntota oblicua: lim ⁡ x→−∞( x x2 − 1) = −∞ (−∞)2 − 1 = 0−

      lim ⁡ x→+∞( x x2 − 1) = +∞ (+∞)2 − 1 = 0+

  • 6. Corte con los ejes:
    • Eje X: x3 = 0 → x = 0 raíz triple. Se obtiene el punto O(0, 0).
    • Eje Y: el punto O(0, 0).
    • Signo: Si x = 2 → f(2) = 23 22 − 1 = 8 3 > 0(+)
  • 7. Máximos y mínimos relativos:
    • A(−3,−33 2 ), máximo relativo.
    • B(3, 33 2 ), mínimo relativo.

    Monotonía:

    • Creciente: (−∞,−3) ⋃ ⁡ (3, +∞)
    • Decreciente: (−3,−1) ⋃ ⁡ (−1, 1) ⋃ ⁡ (1,3)
  • 8. Puntos de inflexión:
    • O(0, 0), punto de inflexión.

    Curvatura:

    • Convexa (⋃ ⁡ ): (−1, 0) ⋃ ⁡ (1, +∞)
    • Cóncava (⋂ ⁡ ): (−∞,−1) ⋃ ⁡ (0, 1)

3.3.2   Ejercicios
Problema 37

Ayudándote de la gráfica, realiza el estudio de la siguiente función f(x) = x x2 − 4.

Solución:
  • Tipo de función: función racional.
  • Dominio: Dom(f) = ℝ −{−2, 2}.
  • Periodicidad: no es periódica.
  • Simetría: simetría respecto al origen.
  • Asíntotas: verticales (x = −2 y x = 2) y horizontal (y = 0).
  • Puntos de corte: (0,0).
  • Máximos y mínimos: no hay.
  • Monotonía: decreciente (−∞, +∞).
  • Punto inflexión: (0, 0).
  • Curvatura:
    • Convexa: (−2, 0) ⋃ ⁡ (0, +∞)
    • Cóncava: (−∞,−2) ⋃ ⁡ (0, 2)

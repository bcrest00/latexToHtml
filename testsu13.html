7.4   Funciones racionales

7.4.1   Modelo de función racional

Analiza y representa la función y = x3 x2 − 1

  • 1. Tipo de función: racional.
  • 2. Dominio: por ser una función racional, hay que excluir las raíces del denominador: x2 − 1 = 0 → x2 = 1 → x 1 = −1, x2 = 1.
    Dom(f) = ℝ −{−1, 1} = (−∞,−1) ⋃ ⁡ (−1, 1) ⋃ ⁡ (1, +∞)
  • 3. Continuidad: es discontinua en x = −1, x = 1, donde presenta discontinuidades de 1ª especie de salto finito.
  • 4. Periodicidad: no es periódica porque las funciones racionales nunca lo son.
  • 5. Simetrías: f(−x) = (−x)3 (−x)2 − 1 = −x3 x2 − 1 = − x3 x2 − 1
    Se observa que f(−x) = −f(x) → función impar → simetría respecto al origen O(0, 0)
  • 6. Asíntotas:
    • Verticales: son las raíces del denominador, x = −1, x = 1
      Posición de la curva respecto a las asíntotas verticales:
      lim ⁡ x→−1−( x3 x2 − 1) = (−1−)3 (−1−)2 − 1 = −1 0+ = −∞

      lim ⁡ x→−1+( x3 x2 − 1) = (−1+)3 (−1+)2 − 1 = −1 0− = +∞

      lim ⁡ x→1−( x3 x2 − 1) = (1−)3 (1−)2 − 1 = 1 0− = −∞

      lim ⁡ x→1+( x3 x2 − 1) = (1+)3 (1+)2 − 1 = 1 0+ = +∞

    • Horizontales: no tiene.
    • Oblicuas: y = x
      Posición de la curva respecto de la asíntota oblicua: lim ⁡ x→−∞( x x2 − 1) = −∞ (−∞)2 − 1 = 0−

      lim ⁡ x→+∞( x x2 − 1) = +∞ (+∞)2 − 1 = 0+

  • 7. Corte con los ejes:
    • Eje X: x3 = 0 → x = 0 raíz triple. Se obtiene el punto O(0, 0).
    • Eje Y: el punto O(0, 0).
    • Signo: Si x = 2 → f(2) = 23 22 − 1 = 8 3 > 0(+)
  • 8. Máximos y mínimos relativos:
    x2(x2 − 3) = 0 → x = 0raíz doble; x = −3, x = 3 raíces simples.
    f″ ⁡(−3) = −33 2 < 0 → A(−3,−33 2 ), máximo relativo.
    f″ ⁡(3) = 33 2 > 0 → B(3, 33 2 ), mínimo relativo.
    Monotonía:
    f′(x) = x4 − 3x2 (x2 − 1)2 → Si x = 2 → f′(2) = 24 − 3 ⋅ 22 (22 − 1)2 = 4 9 > 0(+)
    Las raíces del denominador son discontinuidades dobles.
  • 9. Puntos de inflexión:
    2x(x2 + 3) = 0 → x = 0 raíz simple.
    f‴ ⁡(x) = −6x4 + 36x2 + 6 (x2 − 1)4 → f‴ ⁡(0) = −6≠0 → O(0, 0), punto de inflexión.
    Curvatura:
    f″ ⁡(x) = 2x3 + 6x (x2 − 1)3 → Si x = 2 → f″ ⁡(2) = 2 ⋅ 23 + 6 ⋅ 6 (22 − 1)3 = 28 27 > 0(+)
    Las raíces del denominador son discontinuidades triples.

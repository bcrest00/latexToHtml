3.6   Funciones logarítmicas

Las funciones logarítmicas son funciones cuya ecuación y gráfica pertenecen a uno de los siguientes tipos:

  • f(x) = log ⁡ ax; a > 1

  • f(x) = log ⁡ ax; 0 < a < 1

De la observación y el estudio de las gráficas, se deduce que:

  • Dominio. El dominio de estas funciones es el conjunto de todos los números reales positivos. Dom(f) = ℝ+ = (0, +∞)

  • Monotonía. Estas funciones son:
    • estrictamente crecientes si a > 1.
    • estrictamente decrecientes si 0 < a < 1.
  • Extremos relativos. No tiene.
  • Simetría. Carecen de simetría.
  • Asíntotas. Tienen asíntota vertical.

3.6.1   Modelo de función logarítmica

Analiza y representa la función y = ln ⁡ (x2 − 1)

  • 1. Tipo de función: logarítmica.
  • 2. Dominio: por ser una función logarítmica, el argumento tiene que ser positivo, es decir, mayor que cero.
    x2 − 1 > 0, se resuelve la ecuación correspondiente x2 − 1 = 0 → x2 = 1 → x 1 = −1, x2 = 1. Como las raíces son simples, en cada una de ellas x2 − 1 cambia el signo.

    Dom(f)=(−∞,−1) ⋃ ⁡ (1, +∞)

  • 4. Simetrías: f(−x) = ln ⁡ [(−x)2 − 1] = ln ⁡ (x2 − 1)
    Se observa que f(−x) = f(x) → función par → simetría respecto del eje Y.

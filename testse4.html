Problema 38

Dadas las funciones f(x) = 9x − 5 y g(x) = 4x + 1,

  • f(x) + g(x)
  • f(x) − g(x)
  • f(x) ∘ g(x)
  • g(x) ∘ f(x)
Solución:
  • f(x) + g(x) = 13x − 4
  • f(x) − g(x) = 5x − 6
  • f(x) ∘ g(x) = 9(4x + 1) − 5
  • g(x) ∘ f(x) = 4(9x − 5) + 1

Problema 39

Dadas las funciones f(x) = x3 + 2 y g(x) = x2,

  • f(x) ⋅ g(x)
  • f(x) g(x)
  • f(x) ∘ g(x)
  • g(x) ∘ f(x)
Solución:
  • f(x) ⋅ g(x) = x5 + 2x2
  • f(x) g(x) = x + 2 x2
  • f(x) ∘ g(x) = x(x2)3 + 2
  • g(x) ∘ f(x) = (x3 + 2)2

Problema 40

Calcula la función inversa de las siguientes funciones.

  • f(x) = 1 2x − 1
  • g(x) = 5x − 3 x
  • h(x) = 1 3x − 2
Solución:
  • f(x)−1 = 1 + x 2x
  • g(x)−1 = 3 5 − x
  • h(x)−1 = 2x + 1 3x

Problema 41

Dada la función f(x) = x(ln ⁡ (x))2, calcula:

  • Máximos y mínimos relativos.
  • Puntos de inflexión.
Solución:
  • Máximo relativo: (e−2, 4e−2)
  • Mínimo relativo: (1, 0)
  • Punto de inflexión: (e−1,e−1)

Problema 42

Estudia los intervalos de crecimiento y decrecimiento de la función f(x) = (x + 2)2 x + 1

Solución:
  • Crecimiento: (−∞,−2) ⋃ ⁡ (0, +∞)
  • Decrecimiento: (−2,−1) ⋃ ⁡ (−1, 0)

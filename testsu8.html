2.7   Máximos y mínimos relativos y absolutos

Definición 8. Una función tiene un máximo relativo en x = a si para todo x de un entorno de a se verifica que f(a) es mayor o igual que f(x).

Definición 9. Una función tiene un mínimo relativo en x = a si para todo x de un entorno de a se verifica que f(a) es menor o igual que f(x).

Definición 10. Una función tiene un máximo(mínimo) absoluto en x = a si para todo x ded Dom(f) se verifica que f(a) es mayor(menor) o igual que f(x).

En la siguiente función, se puede observar cuales son los máximos y los mínimos relativos.

2.7.1   Ejercicios
Problema 23

Clacula los máximos y los mínimos de las siguientes funciones:

Solución:
  • Máximo: (0, 0)
  • No presenta ni máximos ni mínimos.
  • Máximo: (1, 3)
    Mínimo: (3,−1)

Problema 24

Calcula los máximos y los mínimos de las siguientes funciones:

  • f(x) = x3 − 3x2 + 3

  • f(x) = 3x4 − 4x3

  • f(x) = 3 x2 + 1

Solución:
  • Máximo en x = 0 y mínimo en x = 2.
  • Mínimo en x = 1.
  • Máximo en x = 0.

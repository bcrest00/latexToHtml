7.5   Funciones irracionales

7.5.1   Modelo de función irracional

Analiza y representa la función y = x2 − 4

  • 1. Tipo de función: irracional.
  • 2. Dominio: por ser una función irracional de índice par, el radicando tiene que ser mayor o igual que cero.
    x2 − 4 ≥ 0, se resuelve la ecuación correspondiente x2 − 4 = 0 → x2 = 4 → x 1 = −2, x2 = 2. Como las raíces son simples, x2 − 4 cambia de signo en cada una de ellas.
    Dom(f)= (−∞,−2] ⋃ ⁡ [2, +∞)
  • 3. Continuidad: es discontinua en x = −2, x = 2.
    • Para x = −2, se tiene f(−2) = 0: lim ⁡ x→−2−(x2 − 4) = 0

      lim ⁡ x→−2+(x2 − 4)no existe

      Por tanto, para x = −2, la función tiene una discontinuidad de 2ª especie.

    • Para x = −2, se tiene f(−2) = 0: lim ⁡ x→2−(x2 − 4)no existe

      lim ⁡ x→2+(x2 − 4) = 0

      Por tanto, para x = 2, la función tiene una discontinuidad de 2ª especie.

  • 4. Periodicidad: no es periódica. Las funciones irracionales nunca lo son.
  • 5. Simetrías: f(−x) = (−x)2 − 4 = x2 − 4
    Se observa que f(−x) = f(x) →función par → simétrica respecto al eje Y.
  • 6. Asíntotas:
    • Verticales: no tiene.
    • Horizontales: no tiene.
    • Oblicuas: presenta asíntotas oblicuas en y = x y en y = −x.
  • 7. Corte con los ejes:
    • Eje X: x2 − 4 = 0 → x2 − 4 = 0 → x2 = 4 → x 1 = −2, x2 = 2
      Se obtienen los puntos A(−2, 0) y B(2, 0)
    • Eje Y: no lo corta.
    • Signo: Si x = 3 → f(3) = 32 − 4 = 9 − 4 = 5 > 0 (+)
  • 8. Máximos y mínimos relativos: f′(x) = x x2 − 4 → x = 0∉ Dom(f)
    No tiene ni máximos ni mínimos relativos.
    Monotonía: si x = 3 → f′(3) = 3 32 − 4 = 3 5 > 0 (+)
  • 9. Puntos de inflexión:
    f″ ⁡(x) = − 4 (x2 − 4)x2 − 4
    f″ ⁡(x) nunca se hace cero, por lo tanto no ha puntos de inflexión.
    Curvatura: si x = 3 → f″ ⁡(3) = − 4 (32 − 4)32 − 4 = − 4 55 < 0 (-)

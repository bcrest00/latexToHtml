2.2   Dominio

Definición 2. El dominio de una función real, también llamado dominio de definición o campo de existencia, es el conjunto de los elementos para los cuales la función está definida. Dicho de otra manera, es el subconjunto de números reales que tienen imagen.

Domf = {x ∈ R∕∃ ⁡y = f(x) ∈ ℝ}

  • Domf: dominio de la función. También se puede denotar por Dom(f) o, simplemente, D. Puede ser todo el conjunto de los números reales, o buen un subconjunto de este: Domf ⊆ R.
  • x: número real, perteneciente al dominio de la función, que recibe el nombre de variable independiente.
  • y: número real, perteneciente al conjunto e la imagen de la función, recibe el nombre de variables dependiente. Para obtener su valor se debe aplicar la función f al valor de x : f(x) = y. Para un par de valores concretos (x,y) se dice que y es la imagen de x, y que x es la antiimagen de y.

2.2.1   Cómo calcular el dominio

Para calcular el dominio eliminaremos"de la ecuación aquellos valores que hagan imposible realizar alguna operación matemática.

  • Funciones polinómicas
    El dominio de toda función polinómica es ℝ, ya que al sustituir un número real cualquiera x ∈ R, siempre va a existir f(x).
  • Funciones racionales
    El denominador debe de ser diferente de cero. Suponiendo una función f(x) = P(x) Q(x), si tanto P(x) como Q(x) son polinomios, entonces el dominio Domf = {x ∈ R∕Q(x)≠0}, es decir, el conjunto de valores que no anulan el denominador.
    Cuando simplifiques una función racional, el dominio debe coincidir con el de la función original, y no debes caer en la tentación de recalcularlo.
  • Funciones irracionales
    • Función irracional de índice impar
      En estos casos, la raíz no impone ninguna restricción adicional al dominio, por lo que coincidirá con el del radicando.
    • Función irracional de índice par
      En estos casos la raíz impone que los valores del radicando siempre sean mayores o iguales que cero.
  • Funciones exponenciales
    En estos casos, la exponencial no impone ninguna restricción adicional al dominio, con lo que coincidirá con el dominio del exponente.
  • Funciones logarítmicas
    En estos casos el logaritmo impone que el argumento debe de ser un número positivo.
  • Funciones trigonométricas
    El dominio de las funciones del tipo f(x) = sin ⁡ [g(x)] y f(x) = cos ⁡ [g(x)], es el dominio de g(x). Las funciones del tipo f(x) = tan ⁡ [g(x)] tienen por dominio: Dom(f) = {x ∈ ℝ|g(x)≠π 2 + kπ, k ∈ ℤ}

Problema 3

Calcula el dominio de las siguientes funciones a partir de la teoría explicada previamente:

  • f(x) = 2x 3x − 2
  • f(x) = 6x x2 − 16
  • f(x) = x2 − 1
  • f(x) = 3 −x + 2
  • f(x) = x + 5 x − 7
  • f(x) = x2 + x
  • f(x) = log ⁡ (−2x2 − 10x − 8)
  • f(x) = arccos ⁡ (x + 5)
  • f(x) = ex
  • f(x) = e−5x
Solución:
  • ℝ −{2 3}
  • ℝ −{−4, 4}
  • (−∞,−1] ⋃ ⁡ [1, +∞])
  • (−∞, 2)
  • (−∞,−5] ⋃ ⁡ [7, +∞])
  • (−∞,−1] ⋃ ⁡ [0, +∞])
  • (−4,−1)
  • [−6,−4]
  • ℝ
